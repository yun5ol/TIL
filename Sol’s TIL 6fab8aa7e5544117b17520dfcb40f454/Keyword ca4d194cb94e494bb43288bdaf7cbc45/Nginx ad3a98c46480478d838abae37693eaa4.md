## Setting

<aside>
💡 세팅파일은 `sudo find / -name nginx.conf` 을 통해서 찾을 수 있음

</aside>

```bash
& nano /etc/nginx/sites-available/default
...
& service nginx reload
```

```json
//프록시
server {
        listen 80;
        listen [::]:80;

        server_name 도메인;

        location / {
                proxy_pass http://localhost:3000/;
        }
}
```

```sql
//SPA (React, etc..)
```
